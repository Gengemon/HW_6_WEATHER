стостоит из 3 фрагментов в бэкстеке
1 - список городов
2 - погода в таблице
3 - погода в WEB
при запуске - список городов
при нажати на город в списке открывается таблица с погодой
при нажатии на кнопку ПОДРОБНЕЕ открывается фрагмент с web страничкой города
при выборе галочки "Показывать при открытии" фрагмент с погодой выбранного города будет открываться при открытии приложения
при нажати аппаратной кнопки НАЗАД, зарывается веб, потом таблица, потом список
